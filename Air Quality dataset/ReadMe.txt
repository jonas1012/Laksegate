Data files are organized in folders:

NOx
NO2

In each folder you will find time series per measurement station. Values for NOx and NO2 are hourly means.
ining NO2 changed.

All time series are present as both csv and Excel.

Be aware that there is a gap in the Hvidovre time series in 2021, where we have no measurements.

All file names have this pattern:

COMPOUNDNAME_XXX_STATIONCODE_STARTYEAR_ENDYEAR.filetype

Station codes are:
HCAB: HC Andersens Boulevard, Copenhagen
HCØ: H.C. Ørsteds Instituttet, Copenhagen
JAGT: Jagtvej, Copenhagen
HVID: Fjeldstedvej, Hvidovre
